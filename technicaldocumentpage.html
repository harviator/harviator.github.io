HTML PAGE STRUCTURE

The basic HTML page structure looks like this:

  • <!DOCTYPE html>
    • <html>
      • <head>
      • </head>
      • <body>
      • </body>
    • </html>

The <!DOCTYPE html> tag tells web browsers this is an html document.

The highest level tag that contains anything is <html></html>.

The <head></head> tag is like the command center of the html file. It defines the document and contains the metadata.

Lastly, the <body></body> tags contain the main content of the website.

BASIC HTML TAGS

There are some basic HTML tags that are good to know.

  • To make a comment you <!--The note you want to leave--> the comment tag is useful to for making notes for yourself or other developers. They can also be used to block code.
  • <header></header> tags usually have information that is on every page such as the navigation menu and branding information.
  • <main></main> tags contain the main content such as articles.
  • <footer></footer> tags are for bottom of the page stuff

While the comment tags can be used anywhere the other tags are typically used within the <body></body> tags using indenting to keep everything neat and organized.

CSS

CSS is used to style elements. You can style elements inline for example, <p style="color: blue;">. You can also use CSS selectors to style elements. To do this you can create a style block within the head of the document.

It would look something like this:

  • <head>
    • <style>
      • p {
        • color: blue;
      • }
    • </style>
  • </head>
